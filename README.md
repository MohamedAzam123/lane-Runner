# 🎮 Endless Runner

**Endless Runner** is a simple yet intelligent endless runner game built with Python and Pygame.
The game features an AI agent that automatically plays by making real-time decisions to avoid obstacles.

---

## 🚀 Project Idea

Endless Runner is a 3-lane arcade-style game where the objective is to survive as long as possible by avoiding incoming obstacles.

Instead of manual control, the game introduces a unique twist:

🤖 **An AI agent plays the game automatically**

---

## 🧠 AI Logic

The game uses a **Utility-Based AI** approach to make decisions:

* Each lane is evaluated based on its "safety level"
* The AI calculates danger depending on:

  * Distance to obstacles
  * Obstacle positions in each lane
* It selects the lane with the highest utility (lowest risk)

📌 In simple terms:

> "Choose the safest lane to survive longer"

---

## 🎮 Game Mechanics

* 🧱 Obstacles spawn randomly across lanes
* ⚡ Game speed increases as the score increases
* 💥 Collision results in Game Over
* 🧮 Score increases when obstacles are successfully avoided

## ▶️ How to Run

1. Make sure Python is installed
2. Install Pygame:

```bash
pip install pygame
```

3. Run the game:

```bash
python main.py
```
